The movie website

The movie website is a multipage website developed by me, which uses html,css and react . this website is my bachelor's degree final project which i got 20 out of 20 for it. i spent almost 3 weeks creating and styling it. My purpose was to create a movie website that shows the lates and toprate dmovies and series like IMDB. I'v used styled components, global style, bootstrap framework and material ui and animations are powered by framer motion. This website also benefits from redux as state manager. The data that is shown in the movie website is pulled from tmdb api (you need to have vpn so pictures can load from tmdb).
